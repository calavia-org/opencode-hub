# Lookup: Agent Reference

**Purpose**: Quick reference for all 18 agents

| Agent | Language | Role | Description |
|-------|----------|------|-------------|
| spec-driven | - | Orchestrator | SPEC-driven development |
| opencode-implementer | - | Implementer | OpenCode/Spec code |
| go-implementer | Go | Implementer | Implements Go code |
| go-tester | Go | Tester | Runs Go tests |
| go-verifier | Go | Verifier | Validates non-functional |
| go-deployer | Go | Deployer | Deploys to Docker/K8s |
| java-implementer | Java | Implementer | Implements Java code |
| java-tester | Java | Tester | Runs JUnit tests |
| java-verifier | Java | Verifier | Validates non-functional |
| java-deployer | Java | Deployer | Deploys to Docker/K8s |
| python-implementer | Python | Implementer | Implements Python |
| python-tester | Python | Tester | Runs pytest |
| python-verifier | Python | Verifier | Validates non-functional |
| python-deployer | Python | Deployer | Deploys to Docker/K8s |
| terraform-implementer | Terraform | Implementer | Writes TF code |
| terraform-tester | Terraform | Tester | Tests + security scans |
| terraform-verifier | Terraform | Verifier | TFLint, checkov, OPA |
| terraform-deployer | Terraform | Deployer | Deploys to TF Cloud |
