# Calavia OpenCode Hub

Centralized OpenCode configuration for `calavia-org` members.

## Quick Start

```bash
# One-time setup
curl -sL https://opencode.calavia.org/setup.sh | bash

# Authenticate with GitHub
opencode auth login

# Start
opencode
```

- This hub is restricted to **calavia-org** organization members
- Authentication via GitHub OAuth

## Available Components

- **Agents**: spec-driven (SPEC-driven development workflow)
- **Modes**: spec-driven
- **Skills**: spec-driven, github-workflow, root-cause-analysis, repo-bootstrap, context7
- **Commands**: spec

## Setup

1. **Clone**: `curl -sL https://opencode.calavia.org/setup.sh | bash`
2. **Authenticate**: `opencode auth login` → select GitHub
3. **Start**: `opencode`
