# Root Cause Analysis Skill

Goal: Find most probable production issue cause quickly

Steps:
1. reproduce - try to reproduce the issue
2. inspect logs - examine logs for clues
3. inspect recent changes - check recent deployments/changes
4. rank hypotheses - prioritize likely causes
5. propose smallest validation - suggest minimal test to confirm
6. define rollback - document rollback procedure if needed
