# Repository Bootstrap Skill

Goal: Become productive quickly in a new repository

Steps:
1. Detect stack - identify language, framework, build system
2. Install dependencies - ensure environment is set up
3. Run tests - verify tests pass
4. Summarize architecture - document key components
5. Propose first useful task - suggest a good starting point
