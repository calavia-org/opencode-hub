Calavia OpenCode Hub v3

Centralized OpenCode configuration. Stack: Java/Kotlin, Python, Go + Docker/Portainer/K8s

Quick Start

export OPENCODE_CONFIG_URL=https://opencode.calavia.org/.well-known/opencode.json
opencode

SPEC-Driven Development

# Start SPEC-driven development
opencode --agent spec-driven
Create a spec for user authentication

# Validate
opencode --agent spec-driven
Validate implementation against SPEC.md

# Implement
opencode --agent implementer
