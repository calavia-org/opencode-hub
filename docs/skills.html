Skill Descriptions

spec-driven

Create and manage specifications for features with full GitHub workflow:

  • Create SPEC.md for new features
  • Detect technology and infer appropriate agent
  • Present agent for user approval
  • Track tasks with checkboxes
  • Archive completed SPECs

github-workflow

Automate GitHub operations linked to SPECs:

  • Create issues from SPECs
  • Create feature branches
  • Open pull requests
  • Link PRs to issues

context7

Get up-to-date library documentation:

# Setup
export CONTEXT7_API_KEY="ctx7_your_key"

# Usage examples:
# "How to use useState in React 19?"
# "pydantic v2 model validation"

root-cause-analysis

Find production issues in distributed systems:

  • Container logs analysis
  • Distributed tracing
  • Service dependency mapping
  • Root cause identification
